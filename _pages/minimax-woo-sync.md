## Features

- **Automatic Invoice Creation** — Sync WooCommerce orders to Minimax accounting as issued invoices
- **PDF Generation** — Automatically retrieve invoice PDFs from Minimax
- **Email Delivery** — Send beautifully styled invoice emails to customers with full customization
- **Product Mapping** — Map WooCommerce products to Minimax items for accurate invoicing
- **Credit Notes (Storno)** — Issue credit notes directly from your WooCommerce admin
- **Multi-Country Support** — Works with Minimax HR, SI, and RS

## Requirements

- WordPress 5.8 or higher
- WooCommerce 5.0 or higher
- A Minimax accounting account ([minimax.hr](https://www.minimax.hr), [minimax.si](https://www.minimax.si), or [minimax.rs](https://www.minimax.rs))

## Installation

1. Download the plugin from the [WordPress Plugin Directory](https://wordpress.org/plugins/minimax-woo-sync/)
2. Upload to your WordPress site via **Plugins > Add New > Upload Plugin**
3. Activate the plugin
4. Go to **WooCommerce > Minimax Sync** to configure your connection

## Privacy & External Services

This plugin connects to the following external services:

- **Minimax Accounting API** — to create invoices, retrieve PDFs, and manage accounting data
- **Bytecode Auth Proxy** — to securely authenticate with the Minimax API without exposing credentials in the plugin
